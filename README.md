# 📔 Daily Journal

A calm, private, distraction-free daily journal. Pick a date, write, and it autosaves to your browser. Browse and search past entries. Nothing ever leaves your device.

**Live demo:** https://skrizidev.github.io/daily-journal/

## Pages
- **Write** — date picker, title, body, live word count, autosave on blur
- **Entries** — searchable list of all past entries with delete
- **About** — privacy notes

## Tech
Vanilla HTML / CSS / JavaScript. Entries stored in `localStorage`.
